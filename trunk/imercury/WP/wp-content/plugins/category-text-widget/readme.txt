This widget displays text/html OR banners only for specific categories. It is as simple as 1-2-3. 

== Description ==

This widget can be used to display custom menus, custom ads or custom descriptions on your sidebar. You can find more useful widgets at [shailan.com](http://shailan.com).

== Installation ==

1. Upload the plugin through `Plugins > Add New > Upload` interface.
1. Activate the plugin once uploaded. 
1. Go to Widgets section and drag'n drop it anywhere you like.
1. Remember to save your widget once you change the content.
